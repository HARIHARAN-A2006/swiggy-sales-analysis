# 🍽️ Swiggy Sales Analysis Dashboard (Excel Project)

## 📌 Project Overview

This project is an end-to-end data analysis and dashboard built in Microsoft Excel to analyze Swiggy food delivery data for the year 2025.

The goal is to derive business insights, track KPIs, and visualize trends related to sales, customer behavior, and regional performance.

---

## 🎯 Problem Statement

Analyze Swiggy's sales data to:

* Track overall business performance
* Understand customer preferences (Veg vs Non-Veg)
* Identify sales trends over time
* Evaluate regional performance
* Support data-driven decision making

---

## 📊 Key Performance Indicators (KPIs)

* **Total Sales** → ₹53.01M (overall revenue)
* **Average Rating** → 4.34 (customer satisfaction)
* **Total Orders** → 197.43K (order volume)
* **Average Order Value (AOV)** → ₹268.51
* **Rating Count** → 5.59K (total reviews)

---

## 📈 Dashboard Features

### 1. Time-Based Analysis

* Monthly Sales Trend
* Daily Sales Trend
* Weekly Sales Trend

### 2. Category Analysis

* Sales by Food Type (Veg vs Non-Veg)

### 3. Geographical Analysis

* State-wise Sales (Map Visualization)
* Top 5 States by Sales

### 4. Performance Summary

* Quarterly Sales, Ratings, and Orders

### 5. Interactive Filters

* Month-wise filter
* Restaurant filter
* Category filter

---

## 🔍 Key Insights

* Non-veg orders contribute **~65% of total revenue**
* Sales peak during **weekends (Saturday highest)**
* **February shows a noticeable dip** in sales
* **Karnataka is the top-performing state**
* Stable weekly trend with minor fluctuations
